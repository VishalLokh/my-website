About Me

Hello! I'm Vishal Lokhande, a Network Engineer with expertise in designing, deploying, and maintaining network infrastructures. I hold a Master's degree in Network Engineering from Southern Methodist University, where I gained hands-on experience with a wide range of networking technologies, including switching, routing, network automation, and cloud networking.

Throughout my career, I have worked with various organizations to implement robust network solutions, troubleshoot complex network issues, and optimize network performance. I'm passionate about automating network configurations using tools like Python and Ansible, and I'm always eager to explore new technologies in the fields of networking and cloud infrastructure.

Currently, I'm based in Dallas, TX, and I'm actively looking for opportunities to further grow my skills in network engineering and automation.

My Projects

Here are a few of the projects I have worked on:

  • Automated Network Configuration Management System: Streamlined network operations using Ansible, Python, and CI/CD pipelines to automate configuration deployment across a multi-vendor environment.
  • Express WiFi VLAN Migration: Migrated customers from VLAN 102 to 235 using Linux scripting, automating the process with network automation tools.

  • Cloud Network Simulation: Built a cloud-based network simulation using GNS3, VMs, and containers, focusing on leaf-spine architecture and network automation with Python and APIs.
