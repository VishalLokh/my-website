AWS Automation Project

This portfolio website is part of an AWS automation project where I leveraged various AWS services to streamline the deployment and management process.

  • AWS S3: The website is hosted on Amazon S3, ensuring high availability and performance.
  • AWS CodePipeline: The deployment of the website is automated using AWS CodePipeline, allowing continuous integration and delivery. Any updates committed to the GitHub repository are automatically deployed to the S3 bucket.
  • AWS Lambda: A Lambda function is used to set the correct metadata automatically (`Content-Type: text/html`) for the files uploaded to the S3 bucket, ensuring proper display of the website content.
  • Automation Scripts: I created automation scripts using Python and Bash for managing updates and deployments, reducing manual intervention.

Overall, this project demonstrates my ability to integrate cloud services and automate deployment pipelines for real-world web applications.
